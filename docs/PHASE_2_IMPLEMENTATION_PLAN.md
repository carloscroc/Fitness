   - Add corresponding SQL insert statements
   - Ensure proper database schema mapping

#### Data Structure Requirements
Each exercise must include:
- id: MD5 hash of exercise name
- name: Exercise name
- muscle: Primary muscle group
- equipment: Primary equipment
- image: Unsplash image URL
- video: YouTube tutorial URL
- overview: 1-2 sentence description
- steps: Step-by-step instructions (array)
- benefits: Array of benefits
- bpm: Average heart rate
- difficulty: Beginner/Intermediate/Advanced
- videoContext: Key form tips
- equipmentList: All equipment needed
- calories: Estimated calories burned
